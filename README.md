## run

1. Clone repo first
   ```bash
   git clone https://github.com/azkadev/specta_apis.git
   cd specta_apis
   ```

2. Run with Docker
   ```bash
   docker build . -t specta_apis
   docker run -it specta_apis
   ```
3. Run with dart
   ```bash
   dart pub get
   dart run
   ```

4. run native exe
    ```bash
    dart pub get
    dart compile exe ./bin/specta_apis.dart -o ./index
    chmod 777 ./index
    ./index
    ```
